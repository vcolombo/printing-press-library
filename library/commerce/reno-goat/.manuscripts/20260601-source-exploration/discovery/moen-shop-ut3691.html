Product Description

Crisp edges and uncomplicated style features give the Voss collection an ageless, yet fashion-forward, presence. Each silhouette brings a confident Transitional style to the bath.

Key Features

  • chrome finish is highly reflective for a mirror-like look that works with any decorating style
  • lever handle offers effortless control
  • M-CORE cartridge included with the trim
  • M-CORE 3 series cartridge included
  • built on the Moen M-PACT common valve system, allowing you to update the faucet style in the future without replacing any plumbing
  • complies with Americans with Disabilities Act (ADA) specifications
  • limited lifetime
